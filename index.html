Pet Robot

ESP32 / XIAO ESP32S3

A conversational desktop companion with expressive animated puppy eyes, voice interaction powered by GPT-4o, and real-time emotion-synced facial expressions. It listens, responds, sings, and reacts to your touch.

👁
Expressive Animated Eyes 8 expressions on SSD1306 OLED — happy, sad, curious, surprised & more
🎙
AI Voice Conversation Whisper STT → GPT-4o Chat → OpenAI TTS — full voice loop
✋
Touch Reactions Tap the head for happy chirps, long press to start a conversation
🎵
Music Player Synthesized tones with emotion-synced expressions on a dedicated FreeRTOS core
🧠
Swappable Personalities SD card personality files — playful, calm, zen, or write your own

Agentic Arduino

AI · Autonomy · Edge

An Arduino platform with embedded agentic AI — capable of autonomous decision-making, tool use, and multi-step task execution entirely on the edge. No cloud required for reasoning. The next frontier of embedded intelligence.

🤖
Autonomous Decision Making On-device AI agent loop — perceive, reason, act without cloud
🔧
Tool Use on Hardware Agent can call hardware functions — move, sense, speak, display
🌐
Hybrid Cloud + Edge Falls back to cloud APIs for complex reasoning when WiFi available
⚡
Low-Power Design Optimized for battery-powered, always-on agentic operation
