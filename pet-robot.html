Pet Robot 🤖

A conversational desktop companion with expressive animated puppy eyes, AI-powered voice interaction, and real-time emotion-synced expressions. Built on ESP32 with the Arduino framework.

Features

👁 Expressive Animated Eyes

Puppy-style eyes on 128×64 OLED using FluxGarage RoboEyes. 8 expressions: neutral, happy, listening, talking, sleepy, angry, surprised, love. Auto-blink every 3±2 seconds. Idle look-around during inactivity.

🎙 AI Voice Conversation

Full voice loop: mic records → Whisper STT → GPT-4o chat (with emotion tags) → OpenAI TTS (24kHz PCM) → speaker playback. Eyes sync to each sentence's emotion in real-time.

✋ Touch Reactions

GestureResponse
Tap (head)Happy eyes + chirp
Long press (head)Start listening
Tap (chin)Love eyes + purring
Long press (chin)Play a song

🎵 Music Player

Synthesized tones with warm harmonics. Plays "Twinkle Twinkle Little Star" with emotion-synced expressions. Runs on a dedicated FreeRTOS core so eyes keep animating.

🧠 Swappable Personalities (S3)

SD card personality files — default (warm puppy), playful, calm, or write your own JSON. Change by editing /robot/config.json. No secrets in source code.

📷 Camera (S3 — Future)

Built-in OV2640 camera. Planned: visual context for conversations ("What do you see?"), face detection, presence awareness, photo capture on command.

Architecture

// state machine

IDLE ──tap──→ REACTING
  │
  ├──voice──→ VOICE_REACT
  │             (tracks loudness)
  │
  ├──long press──→ LISTENING
  │                 → THINKING
  │                 → SPEAKING
  │                (emotion eyes)
  │
  └──chin long──→ MUSIC_PLAYING
                  (dedicated core)

// audio pipeline (S3)

PDM Mic (16kHz mono)
  → record buffer (PSRAM)
  → Whisper API → text
  → Chat API
  → emotion-tagged JSON
  → TTS API → 24kHz PCM
  → Resample 44100Hz stereo
  → I2S → MAX98357A → 🔊
